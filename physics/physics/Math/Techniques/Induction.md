Given $P(n)$, a property depending on a positive integer $n$,

1. If $P(n_0)$ is true for some positive integer $n_0$, and
    
2. If for every $k \ge n_0$, $P(k)$ true implies $P(k + 1)$ true,
    

then $P(n)$ is true for all $n \ge n_0$.


Strong Form:

Given $P(n)$, a property that depends on an integer $n$,

1. If $P(n_0), P(n_0 + 1), \dots, P(n_0 + m)$ are true for some positive integer $n_0$ and nonnegative integer $m$, and
    
2. If for every $k > n_0 + m$, $P(j)$ true for all $n_0 \le j < k$ implies $P(k)$ true,
    

then $P(n)$ is true for all $n \ge n_0$.
