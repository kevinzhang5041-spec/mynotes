For $$\sum a_n,$$ compute  
$$L=\limsup_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right|$$  
(or simply the limit if it exists). 

- If $L<1$: series converges absolutely
    
- If $L>1$ (or $=\infty$): series diverges
    
- If $L=1$: inconclusive  
    Tips:
    
- Best for factorials, exponentials, powers, products
    
- After simplifying,
    
    - remaining constant $<1$ $\Rightarrow$ converges
        
    - remaining constant $>1$ $\Rightarrow$ diverges
        
    - limit $=1$ $\Rightarrow$ use another test
