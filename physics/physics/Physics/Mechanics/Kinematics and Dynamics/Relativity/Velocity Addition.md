### Parallel 
When an object is observed by frame $S'$ to have a speed $v_1$, and $S'$ is observed to have a speed $v_2$ by frame $S$, then the object in $S$ has speed
$$
u=\frac{v_1+v_2}{1+v_1v_2/c^2}
$$
### Transverse
An object moves with velocity $(u'_x, u'_y)$ with respect to frame $S'$ which in turn moves with speed $v$ with respect to frame $S$. The $y$ component of the velocity of the object in $S$ is given by
$$
u_y = \frac{u_y'}{\gamma(1+u_x'v/c^2)}
$$
