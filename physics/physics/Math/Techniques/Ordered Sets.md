An order on a set is a relation $\le$ with three properties:

1. $a \le a$
    
2. If $a \le b$ and $b \le a$, then $a = b$
    
3. $a \le b$ and $b \le c$ implies $a \le c$
    

The order is called **total** if any two elements are comparable, that is, if for every $a$ and $b$, either $a \le b$ or $b \le a$. The simplest example of a total order is $\le$ on the set of real numbers. The existing order on a set can be found useful when one is trying to solve a problem.

An order on a finite set has maximal and minimal elements. If the order is total, the maximal (respectively, minimal) element is unique.
